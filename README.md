## 主要功能介绍

1.自动换号登陆

​	帐号密码文件保存在手机（模拟器）本地端~\Picture\account.txt

​	格式如下：

```
username----password//注意！以四个‘-’（减号）分隔

比如：
sakura----123456
ilovepcr----654321
```

2.自动抽取免费十连

3.自动收取礼物

4.自动收取任务奖励

5.自动收取工会之家体力（mana、扫荡券等）

6.自动扫荡H本

​	这里是从H1-1开始依次一个个扫过去（用扫荡券）

7.自动扫荡探索本

​	默认选择星级最高的关卡（用扫荡券）

8.自动手工1-1（囤积扫荡券）//暂时还没做，应该很快就会做完
